## Running Web App

```bash
# Install yarn
curl --compressed -o- -L https://yarnpkg.com/install.sh | bash

# Install typescript, angular cli
yarn global add typescript @angular/cli

# In root directory install node_modules
yarn

# Serve the angular app
ng serve
```
