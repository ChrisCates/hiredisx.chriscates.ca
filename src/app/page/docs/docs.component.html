hiredisx::set

bool hiredisx::set(std::string key, std::string value)

hiredisx::set returns a boolean after setting a key. If true, this means that the key was successfully set. If false, this means the key was not set. The key that was set is now retrievable via the key string specified.

Example

hiredisx::get

std::string hiredisx::get(std::string key)

hiredisx::get returns the string value of a specified key. If the key has not been set it will simply just return an empty string. If there is an error connecting with the database. The string value will contain details of why it failed.

Example

hiredisx::del

bool hiredisx::del(std::string key)

hiredisx::del deletes a specified key. It will return true if the key has been successfully deleted. It will return false if there was an error connecting with the database.

Example

hiredisx::push

bool hiredisx::push(std::string key, std::string value, std::string prefix = "L")

hiredisx::push appends a value to a list. If the list does not exist, it will create a new one. Note that it works similar to the set command, however, it includes a prefix that is default to "L". The prefix can be either "L" or "R". If left, it will prepend to the beginning of the list. If "R" it will append it to the end of the list.

Example

hiredisx::list

std::vector<std::string>

list(std::string key, int start = 0, int end = -1, std::string prefix = "L")

hiredisx::list retrieves a vector of the list. The start integer specifies where to start in the list. The end integer specifies where to stop. If the end integer is set to -1. It will retrieve til the end of the list. Similar to the push command. The prefix "L" means to iterate forward from start to finish. While "R" means to iterate backwards from finish to the starting point.
